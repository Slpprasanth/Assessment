# Assessment – React Native & Node.js

## Overview
This is a one-page application where users can enter items one at a time and view them as a list.

## Approach & Planning
- React Native used for frontend UI
- Node.js used for backend APIs
- Simple and clean structure followed
- Focus on clarity and assessment requirements

## Technology Stack
- React Native – Frontend UI
- Node.js – Backend runtime
- Express.js – REST API framework

## Project Structure
backend/
  server.js
frontend/
  App.js
README.md

## Execution
- User enters item in input field
- Clicks Add button
- Item appears in a numbered list below a title
